# Introduction

AI-Map is an application for creating and maintaining mappings from local, organization-specific
code systems to internationally recognized standards.

Typical targets include:

- **SNOMED CT** — clinical concepts, findings, diagnoses, and procedures
- **LOINC** — laboratory tests and clinical observations

The goal of this documentation is to provide an operational guide to the mapping workflow,
data quality expectations, and governance practices.

!!! info "Documentation version"
    This documentation aligns with **AI-Map 0.7.0**.
